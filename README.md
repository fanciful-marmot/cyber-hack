# Cyber Hack
Replica of Cyberpunk 2077 hacking minigame. Still a work in progress

Just a little project with React.js and Redux.

## Setup
 - Install node.js and npm
 - Run `npm install`
 - Run `npm start` to run the dev build and dev server
